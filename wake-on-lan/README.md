# ⏰ Wake-on-LAN (WoL)

O serviço de **Wake-on-LAN** permite "acordar" computadores desligados na rede local através do envio de um pacote "mágico" (Magic Packet).

---

## ⚙️ Uso no pfSense
1.  Cadastrar o endereço MAC e o Hostname do dispositivo.
2.  Clicar no ícone de "Acordar" na interface web.

---

## 🛠️ Template XML
O arquivo `template_wol.xml` contém a estrutura para cadastrar um host no serviço WoL.
